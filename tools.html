Tools

In the package are avaliable some tools to help the user to work with XML files and strings.

SimbioReader.tools.camel_case(text) → str[source]

Converts a given string to camelCase.

The function transforms a string by removing underscores or hyphens, capitalizing the first letter of each word except the first one, and then joining them together to form a camelCase string.

Parameters:

text (str) – The input string, typically in snake_case or kebab-case.

Returns:

The converted string in camelCase.

Return type:

str

Example

>>> camel_case('hello_world')
'helloWorld'
>>> camel_case('hello-world')
'helloWorld'

SimbioReader.tools.snake_case(text) → str[source]

Converts a given string to snake_case.

The function transforms a camelCase or PascalCase string to snake_case, replacing capital letters with underscores followed by the lowercase equivalent.

Parameters:

text (str) – The input string, typically in camelCase or PascalCase.

Returns:

The converted string in snake_case.

Return type:

str

Example

>>> snake_case('helloWorld')
'hello_world'
>>> snake_case('HelloWorld')
'hello_world'
